# How It Works
Simple. Intuitive. Powerful.
1️FARMER INPUT
   └─> Upload fish photo or describe observations via voice
       "My fish are swimming near the surface"

2️ AI ANALYSIS  
   └─> Computer vision analyzes behavior, skin lesions, abnormalities
       ⏱️ Analysis complete in 15 seconds

3️ RISK SCORE
   └─> ML models generate risk scores (Low/Medium/High/Critical)
       ✓ 85% accuracy in early detection

4️ ALERTS & ACTION
   └─> Instant SMS alerts with specific treatment suggestions
       "Add lime to pond. Check in 6 hours."

## Quick Start

### Prerequisites
- Python 3.9+
- Node.js v16+
- MongoDB or PostgreSQL

### Installation

```bash
# Clone the repository
git clone https://github.com/Git-Ninja-afk/Finguard.git
cd Finguard

# Install AI/ML dependencies
cd ai-models && pip install -r requirements.txt

# Install backend dependencies
cd ../api && npm install

# Install frontend dependencies
cd ../dashboard && npm install

# Start all services
npm run dev
```
